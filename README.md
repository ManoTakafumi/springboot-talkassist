# Talk Assist

会話に不安を感じる方に向けて**無難に会話への取っ掛かりを作りサポートするWebアプリ**です。
雑談や挨拶が苦手な人でも、場面やキーワードを入力するまたは入力せずにランダムで自然な一言を得られることを目的にしています。

---
## アプリ概要

- キーワードを入力すると、関連する会話例を1つ提案します。
- キーワード未入力ではランダムでの表示を行います。
- 同じキーワードで他の会話例が欲しい場合は、ボタン一つで同じキーワードでの提案が可能です。
- 会話データはMySQLでの永続的に保存可能です。

「考えることが苦手でも会話に活し、使える」ことを目的として設計しました。

---
## 開発背景・目的

日常会話や職場での雑談に対しての苦手意識を持つ人は一定数いると思います。
「正解がない会話」に対しての不安払拭の一助になりたいと思い考えました。
本アプリは、
- 話しかけることを助ける
- 無難で失礼の少ない会話を素早く提示する。
ことで、会話への心理的ハードルを下げることを目的としています。

---
## 主な機能

### 1.話題検索機能
- キーワード（例：朝、挨拶、休憩）を入力すると該当する話題を表示
- キーワードは部分一致に対応

### 2.ランダム提案機能
- キーワード未入力時に、登録されている話題からランダムで１件表示
- ワンクリックするだけで話題の提供が可能

### 3.再検索導線
- 提案画面からキーワードの再入力が可能
- 入力画面への「戻る」操作を減らし、UXを重視

## データ設計について
- 会話例データはMySQLに永続化
- data.sqlは初期データ投入用に使用
- データは削除せず、重複についてはコマンドによる削除で対応
運用も想定し、データの紛失を避けるように意識しています。

---
## 設計・実装で意識した点
- 入力必須ではなく、未入力時でも利用できる設計
- フロント側ではなく、サーバー側で処理を行う
- データが増えても対応できるように設計
